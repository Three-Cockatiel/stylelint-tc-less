# stylelint-tc-less
`styleslint-tc-less`基于`stylelint-config-standard`配置，是所有前端开发者公认的前二的styleslint代码规范配置。加上本人认同的一些特质化规则，为less项目服务。

  **目的是tc开源团队的stylelint规范，服务less项目**

# 安装
```javascript
  npm install styleslint-tc-less -D
```

# 使用
```javascript
{
  "extends": ["stylelint-tc-less"]
}
```
